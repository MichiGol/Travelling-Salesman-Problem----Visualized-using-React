# Microdosing Infinity
Interactive visualization tool for the Travelling Salesman Problem built with React.

## Features
- **World Tour**: place real cities on a world map and watch 2-opt optimize the route
- **Algorithm Lab**: run 8 algorithms (Brute Force, Held-Karp, Nearest Neighbor, 
  Cheapest Insertion, 2-opt, MST 2-Approx, Simulated Annealing, Ant Colony) 
  with live animation and step-by-step mode
- English/German language toggle

## Try it
No installation needed: [Live Demo] https://legendary-queijadas-22e0b4.netlify.app/

## Run locally
Requires Node 18+
```bash
npm install
npm run dev
```

## Built with
React, Vite, Tailwind CSS
